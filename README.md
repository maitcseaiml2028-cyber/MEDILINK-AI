# Health Stack Manager

A comprehensive health management application to handle patient appointments, hospital operations, and doctor dashboards.

## Features
- **Patient Management:** Track patient history and schedule appointments.
- **Doctor Dashboard:** Overview of scheduled patients and upcoming appointments.
- **Hospital Administration:** Manage check-ins, check-outs, and staff details.

## Tech Stack
- Frontend: React / Vite / TailwindCSS
- Backend: Node.js / Express
- Database: SQLite / Drizzle ORM
