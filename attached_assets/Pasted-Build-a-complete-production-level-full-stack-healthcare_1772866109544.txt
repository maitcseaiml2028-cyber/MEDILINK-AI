Build a complete production-level full-stack healthcare platform called "MEDLINK-AI". The system should be a modern digital healthcare record and appointment management platform that allows patients, doctors, and administrators to securely manage medical records, appointments, and digital prescriptions.

The project must be fully functional and include frontend, backend, database, authentication, role management, and modern UI design.

TECH STACK

Frontend:
- React.js (Vite)
- Tailwind CSS
- Axios
- React Router
- React Query or Context API for state management
- QR Code generator and QR scanner library
- Chart.js or Recharts for dashboard analytics

Backend:
- Node.js
- Express.js
- MVC architecture
- RESTful API

Database:
- MongoDB with Mongoose

Authentication & Security:
- JWT authentication
- Role-based access control (Patient, Doctor, Admin)
- Password hashing using bcrypt
- Environment variables for secrets
- Input validation and secure API routes

Cloud / Storage:
- File upload system using Multer
- Store medical documents (reports, prescriptions)
- Prepare structure for cloud storage integration

AI Integration:
- AI medical summary generator for patient records
- AI chatbot assistant for doctors
- OCR system for extracting text from uploaded prescriptions
